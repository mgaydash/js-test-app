Element Cloning

    In this example we will see how we can take one element and clone it to produce several, and tie a widget to each one of them. when you click the "Initialize Widgets" button the blue box will be removed and cloned. Then a for loop will append clones of that element to the container div below. Each element will have their own widget tied to them, the widget is very simple for demonstration purposes. The widget counts the number of times that the user clicks the div. Notice how each widget will keep track of its own count even though they are all using the same widget they all have different instances of it.



Event Communication

    In this example we have 3 <div>'s each with a widget called "colorSquareWidget" tied to them that when clicked will choose a random color between red,orange,yellow,green,and blue. The first <div> containing text, has a widget tied to it called "colorCountWidget". Each 'colorSquareWidget' is bound to the "Communicate" button below that when clicked will trigger an event called "communicate" and pass it's current background-color. The colorCountWidget is subscirbed to this "communicate" event and changes its local count variables depending on which colors are passed. With this form of coding we can remove/add as many <div>'s with the "colorSquareWidget" tied to them as we please and the code will remain functional.
